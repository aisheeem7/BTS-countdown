BTS Comeback Countdown – March 20, 2026

A minimal and animated web page created to celebrate BTS’s comeback after a 3-year hiatus on 20th March, 2026.
The website displays a live countdown timer, smooth transitions, and a comeback-themed design that builds excitement as the date gets closer.

This project was built to:

Visually celebrate BTS’s historic return
Create hype using a real-time countdown
Practice front-end animation and UI design
Deliver a fan-friendly interactive experience
